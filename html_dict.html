    Dicionário HTML

    Referência rápida das tags usadas neste projeto.

    Dicionário de Tags HTML

    • <!DOCTYPE html> — Define o tipo de documento como HTML5.
    • <html> — Elemento raiz de um documento HTML.
    • <head> — Contém metadados e links para recursos externos.
    • <meta> — Define metadados como codificação de caracteres e viewport.
    • <title> — Define o título da aba/navegador.
    • <link> — Conecta o HTML a um arquivo externo de CSS.
    • <body> — Contém todo o conteúdo visível da página.
    • <header> — Seção de cabeçalho, geralmente com navegação.
    • <nav> — Contém os links de navegação do site.
    • <menu> — Lista de itens de menu.
    • <li> — Define um item dentro de uma lista.
    • <a> — Define um hiperlink.
    • <div> — Container genérico para agrupamento de conteúdo.
    • <aside> — Conteúdo auxiliar (barra lateral, por exemplo).
    • <img> — Exibe uma imagem.
    • <h2> — Define um subtítulo de nível 2.
    • <p> — Define um parágrafo de texto.
    • <main> — Destaca o conteúdo principal da página.
    • <section> — Agrupa conteúdo relacionado em uma seção.
    • <ul> — Cria uma lista não ordenada.
    • <strong> — Destaca um texto como importante (negrito).
    • <footer> — Rodapé da página.
